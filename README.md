# 📚 Shelfd — Modern Reading Tracker

A sleek, privacy-focused Progressive Web Application (PWA) to track your reading habits with a warm dark theme and detailed reading analytics.

🚀 **Live Demo**: [https://nzewail.github.io/bookshelf/](https://nzewail.github.io/bookshelf/)

---

## ✨ Features

- 📕 **Format Tracking**: Track books by format — **📕 Physical**, **🎧 Audio**, and **📱 Ebook**.
- 📌 **Status Management**: Easily group books into **📖 Currently Reading**, **✨ Want to Read**, **✅ Finished**, and **🚫 DNF (Did Not Finish)**.
- 📊 **Detailed Reading Analytics**:
  - **Yearly Goal Ring**: Track your progress towards your annual reading goal.
  - **Books Per Month**: Interactive monthly chart with numerical Y-axis and drill-down book lists.
  - **Book Length Breakdown**: Page length buckets (`<200`, `200–300`, `300–400`, `400–500`, `500+` pgs).
  - **Format & Genre Distribution**: Visual percentages for reading formats and Fiction / Non-Fiction genres.
  - **DNF Stat Card & Page Tracking**: DNF pages read count toward total pages, with dedicated DNF metrics.
- 🔍 **Open Library Search**: Search millions of books by title or author with automatic cover fetching.
- 🊠 **Flexible Layouts**: Switch between **Standard Grid**, **Compact Grid**, **Large Cover View**, and **List View**.
- 🗂️ **Collapsible Shelf Sections**: Expand or collapse sections with persistent state memory.
- 📱 **Progressive Web App (PWA)**: Installable as a native-feeling app on iOS and Android with offline caching.
- 🔒 **Privacy & Data Ownership**: 100% client-side (`localStorage`) with full JSON backup **Export & Import**.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript (ES6+), CSS Custom Properties (Variables)
- **API**: Open Library Search & Covers API
- **Storage**: Browser `localStorage` with JSON export/import
- **Hosting**: GitHub Pages

---

## 📱 Mobile Installation (PWA)

1. Open [https://nzewail.github.io/bookshelf/](https://nzewail.github.io/bookshelf/) on your mobile browser.
2. **iOS (Safari)**: Tap the **Share** icon $\rightarrow$ **Add to Home Screen**.
3. **Android (Chrome)**: Tap **⋮** $\rightarrow$ **Install App**.
